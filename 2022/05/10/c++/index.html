友元

友元可以访问类内所有成员

class Animal;
class Cat;
class Cat:protected Animal{
};

class Cat final{
};

class Animal{
protected:
void jump() {
cout << “跳” << endl;
}
};
class Cat:protected Animal{
public:
using Animal::jump;//修改jump权限为public
}

<font color="#FF66CC">private成员永远只能被它内部的成员访问</font>  
因此不允许派生类使用using改变private成员的访问权限  
注：由于继承导致的权限变化不算入其中
